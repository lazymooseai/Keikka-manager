# 🎸 Keikkasetti Manager

Streamlit-pohjainen sovellus keikkasetlistan hallintaan.

## Ominaisuudet
- Setlistan syöttö TXT tai PDF-tiedostona
- Biisikortit PDF-viewerillä
- Sävellaji- ja transponointi-info
- Google Drive -integraatio (tulossa)
- Automaattinen sointujen transponointi (tulossa)

## Asennus

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Setlistan formaatti

```
1. Biisinimi | Sävellaji
2. Biisinimi | Alku -> Kohde
3. Biisinimi | G (capo 2)
```
